# ⚡ 一键部署 PWA — 2 分钟拿到 HTTPS URL（带离线缓存）

> **核心约束**：手机端要"离线缓存"必须走 **HTTPS**（Service Worker 的硬性要求）。
> 局域网 `http://192.168.1.6:8766` 是 HTTP，不工作。
> 所以必须把应用放到一个有 HTTPS 的地方。

下面给你**最简单的两条路**，按推荐度排序：

---

## ⭐ 推荐方案 · Netlify Drop（拖一下就完事，不用注册）

### 步骤

**① 准备好打包文件**：
桌面已经生成 `火山售前学习PWA.zip`（274 KB），就是要上传的内容。

> 完整路径：`C:\Users\11498\Desktop\火山售前学习PWA.zip`

**② 打开 Netlify Drop**（电脑浏览器，不要用手机）：

🔗 **https://app.netlify.com/drop**

**③ 把 zip 拖进网页中央的虚线框**

> 注意：要拖 **解压后的文件夹**，不要直接拖 zip。
> 操作建议：先双击 zip 解压到桌面（变成一个文件夹），然后把**解压出来的文件夹本身**拖进 Netlify Drop 的虚线框。
>
> 或者更简单：**全选文件夹里所有文件 → 一起拖进虚线框**。

**④ 等 30 秒**：
Netlify 自动构建 + 分配 HTTPS URL。会跳到一个页面，类似：

```
https://random-words-12345.netlify.app
```

✅ 这就是你的 PWA 永久地址（Netlify 不注册账号也能保留 24 小时；注册后永久保留）。

**⑤ 手机访问 + 添加到主屏幕**：

- 手机**任意网络**（4G/WiFi 都行）打开浏览器
- 输入上面的 URL
- 浏览器**自动弹出"安装应用"** → 点同意
- 桌面出现 **VE 蓝紫渐变图标** → 点开就是全屏 App
- **第一次启动后断网也能用**（Service Worker 在 HTTPS 下激活）

### 想保留 URL 不变？

部署完页面里有**"Claim site"**按钮 → 用 GitHub/Google/Email 注册 → 站点永久绑到你的账号。
还能改成自定义 URL（如 `volcengine-learn.netlify.app`）。

## 🆘 常见问题

**Q：拖上去后页面不显示，提示找不到 index.html？**
A：拖了文件夹本身（外层有一层多余目录）。重新解压 zip，**进入解压出来的文件夹**，全选里面的文件再拖。

**Q：Netlify 临时 URL 想保留怎么办？**
A：部署后页面顶部有 "Claim this site"，登录就锁定。

**Q：我能用其他平台吗？**
A：完全可以，逻辑一样：
- **Cloudflare Pages**（pages.cloudflare.com）— 免费、CDN 极快
- **Vercel**（vercel.com）— 拖拽部署同样支持
- **GitHub Pages**（见 GitHub 那份指南）

**Q：手机访问后没有"安装应用"提示？**
A：① 确认是 HTTPS（URL 以 https:// 开头）② 浏览器菜单里手动选 "添加到主屏幕"

**Q：下次想更新内容怎么办？**
A：改完代码后用 PowerShell 重新打包：
```powershell
Compress-Archive -Path "E:\ai\claude\火山引擎学习\index.html",
"E:\ai\claude\火山引擎学习\manifest.webmanifest",
"E:\ai\claude\火山引擎学习\sw.js",
"E:\ai\claude\火山引擎学习\icons",
"E:\ai\claude\火山引擎学习\scripts",
"E:\ai\claude\火山引擎学习\styles" `
  -DestinationPath "$env:USERPROFILE\Desktop\火山售前学习PWA.zip" -Force
```
然后回到 Netlify 站点页面，找到 "Deploys" → "Drag and drop your site folder here" 重新拖一次即可。

> ⚠️ 改 `sw.js` 时记得 bump 版本号（`CACHE_VERSION = 'volcengine-presales-v2'`），否则手机端可能继续用旧缓存。
