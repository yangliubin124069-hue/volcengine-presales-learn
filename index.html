📲 怎么把应用装到桌面

不同浏览器步骤不同，照下面对应的做：

🌐 Chrome / Edge
  1. 右上角 ⋮ 菜单
  2. 选 "添加到主屏幕" / "安装应用"
  3. 确定 → 桌面出现 VE 图标，点开就是全屏 App
🦊 Via 浏览器
  1. 右下角 三横菜单
  2. 选 "添加到桌面" → 类型选 "应用"
  3. 授予权限提示出现时同意
🐲 华为浏览器
  1. 底部 ≡ 菜单
  2. 选 "添加到桌面"
  3. 注：华为浏览器只能加快捷方式（带地址栏），但功能完整可用
🐧 QQ 浏览器 / 夸克
  1. 菜单 → "添加到桌面" 或 "网页应用"

如果某个步骤不见 → 切换到 Via 浏览器 或 Chrome 重试
